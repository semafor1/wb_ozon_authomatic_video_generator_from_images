Как запустить скрипт (WB/Ozon видео из фото)

Файл скрипта: marketplace_video_maker.py
Папка проекта: <путь_к_папке_проекта>

========================================
1) Первый запуск на НОВОМ ПК
========================================

Шаг 1. Установите Python 3.10+ (если не установлен)
1. Скачайте Python с официального сайта: https://www.python.org/downloads/windows/
2. При установке включите опцию "Add python.exe to PATH".
3. Проверьте в PowerShell:
   python --version

Шаг 2. Установите ffmpeg
1. Скачайте ffmpeg для windows https://github.com/BtbN/FFmpeg-Builds/releases
2. Распакуйте ffmpeg, например, в: C:\ffmpeg
3. Добавьте в PATH папку: C:\ffmpeg\bin
4. Проверьте:
   ffmpeg -version

Если ffmpeg не найден, выполните в PowerShell (для текущего пользователя):
$ff = "C:\ffmpeg\bin"
$userPath = [Environment]::GetEnvironmentVariable("Path", "User")
if ($userPath -notlike "*$ff*") {
  [Environment]::SetEnvironmentVariable("Path", ($userPath.TrimEnd(';') + ";" + $ff), "User")
}
После этого перезапустите PowerShell/IDE.

Шаг 3. Откройте папку проекта
В PowerShell:
cd "<путь_к_папке_проекта>"

Шаг 4. Запустите скрипт
python marketplace_video_maker.py

Шаг 5. В интерфейсе
1. Выберите маркетплейс (wildberries / ozon / wildberries и ozon).
2. Добавьте папки с фото:
   - вставьте пути вручную в поле, или
   - нажмите "Добавить папки (мультивыбор)".
3. При необходимости задайте:
   - переходы,
   - максимальную длительность,
   - минимальное время на 1 фото,
   - FPS.
4. Нажмите "Собрать видео".

Где будут файлы:
- Видео сохраняются в той же папке, где лежат фото.
- Имена вида: <имя_папки>_wildberries.mp4 и/или <имя_папки>_ozon.mp4

========================================
2) Повторный запуск на ПК, где уже запускали
========================================

Шаг 1. Откройте PowerShell в папке проекта:
cd "<путь_к_папке_проекта>"

Шаг 2. Запустите:
python marketplace_video_maker.py

Если не запускается:
1. Проверьте Python:
   python --version
2. Проверьте ffmpeg:
   ffmpeg -version
3. Если ffmpeg не найден, добавьте C:\ffmpeg\bin в PATH (см. блок выше) и перезапустите PowerShell.

========================================
3) Быстрая проверка перед работой
========================================

1. Одна команда проверки:
   python --version
   ffmpeg -version
2. Убедитесь, что есть хотя бы одна папка с фото (.jpg/.jpeg/.png/.webp/.bmp/.tif/.tiff).
3. После сборки проверьте, что итоговый .mp4 создался в папке с фото.
